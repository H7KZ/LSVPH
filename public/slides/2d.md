## Rastrová vs. vektorová grafika

|          | Rastrová       | Vektorová  |
| -------- | -------------- | ---------- |
| Příklad  | PNG, JPG       | SVG, AI    |
| Zvětšení | Ztrácí kvalitu | Bez ztráty |
| Použití  | Textury, fotky | Loga, UI   |

## Herní asssety

- Sprite — obrázek jednoho herního objektu
- Spritesheet — více spritů v jednom souboru
- Tile — dlaždice pro skládání herního světa
